# Obtaining API Keys

## WeatherAPI

Register at [https://www.weatherapi.com/signup.aspx](https://www.weatherapi.com/signup.aspx) to obtain a key.

![image](https://github.com/user-attachments/assets/6b75aedc-7006-4ec3-9c9b-aa8be42f72c4)

No payment card is required.

## AeroDataBox API

1. Register on API.market at [https://api.market/auth/login](https://api.market/auth/login) to obtain a key.

![image](https://github.com/user-attachments/assets/e6384f60-96f4-4b2c-8743-9cef0629a71f)

2. Then subscribe to the Basic plan for AeroDataBox API at [https://api.market/store/aedbx/aerodatabox](https://api.market/store/aedbx/aerodatabox). No payment card is required.

![image](https://github.com/user-attachments/assets/1d5656c8-bf74-44ce-b7ba-ef273000111c)

**Unfortunately, on February 12, 2026, the Free plan was removed from API.market.** You now need to buy the Pro plan (6,000 units) or choose FlightAware API.

## FlightAware API

1. Register for FlightAware AeroAPI at [https://login.flightaware.com/signup](https://login.flightaware.com/signup).

2. On the [Account and billing](https://www.flightaware.com/aeroapi/portal/account) page, enter your payment card details and choose the **AeroAPI Personal** plan. This plan includes $5 of free usage, which is usually enough for personal use.

3. Your API key is available on the [main portal page](https://www.flightaware.com/aeroapi/portal/).
